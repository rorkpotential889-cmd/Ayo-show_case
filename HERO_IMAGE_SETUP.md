# Hero Section Image Setup

## Add Your Hero Image

Place your professional developer image in:

```
public/hero/developer.jpg
```

**Image Requirements:**

- Format: JPG, PNG, or WebP
- Recommended Size: 1200x900 or larger
- Aspect Ratio: 4:3 or 16:9 works best
- File Size: Keep under 1MB for fast loading

## Current Setup

✅ Hero section is ready to display your image
✅ Fallback gradient will show until image is added
✅ Image will automatically appear once placed in the correct location

## How to Add

1. Save your developer/professional image
2. Name it exactly: `developer.jpg`
3. Place it in: `public/hero/` folder
4. Refresh your browser - image will appear!
